ShortenUrl

List shorten url

HTTP GET

Endpoint Description
http://localhost:888/api/find Used to list and search through out all the available shorten url

Endpoint Parameters

Parameter Required Type Default Description
sort_by X String(count) Null Sorts the results by choosen value
limit X Integer 40 The limit of results per page that has been set

Create shorten url

HTTP POST

Endpoint Description
http://localhost:888/api/create Create shorten url
